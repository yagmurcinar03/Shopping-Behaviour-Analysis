# Shopping Behavior Analysis

## Overview
This project analyzes customer shopping behavior using demographic, transactional, and product-related data to uncover patterns in purchasing decisions.

## What I Did
- Analyzed customer purchase behavior and spending patterns
- Examined relationships between purchase amount and factors such as age, subscription status, and promo code usage
- Used data visualization techniques (scatter plots, box plots, treemaps) to identify trends
- Generated insights to support business decisions and marketing strategies

## Key Insights
- Customers tend to spend more on items they previously purchased
- Subscription status affects consistency in purchasing behavior
- Review ratings show a positive relationship with purchase amount
- Seasonal trends influence product demand and spending patterns
